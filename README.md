## About The Project

 myBooks is an e-commerce website where a user can upload there own books and purchase books of others.
 
 <a href="https://loveandcare.herokuapp.com/">Deploy Link</a>

### Built With

* Node.JS
*	Express
*	Node Package Manager
*	Mongodb
*	Mongoose
*	Javascript
*	Template engine.
* DOM manipulation.
*	HTML-5
*	CSS/Bootstrap
*	Handlebar
*	Express file-upload.
*	Cloudinary
*	Heroku

 ## Getting Started

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/attainu/backend-project-hari-elangeswaran-au16.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run server.js 
   ```sh
   npm start
   ```
